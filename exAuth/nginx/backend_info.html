Protected Backend Service

This is a sample backend service protected by JWT authentication.

Only requests with valid JWT tokens should reach this page!

Authentication Information Received:

Customer ID: $http_x_customer_id
Account ID: $http_x_account_id
User ID: $http_x_user_id

Request Details:

Request URI: $request_uri
Request Method: $request_method
Remote Address: $remote_addr
User Agent: $http_user_agent

Available Endpoints:

  • / - Main page (shows auth info)
  • /api/ - API endpoint (shows auth info in JSON)
  • /headers - All received headers (for debugging)
  • /health - Health check
